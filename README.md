# vue-demi-sample

This template should help get you started developing with Vue 2/3 in Vite.


## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev:2.7
pnpm dev:3
```

### Compile and Minify for Production

```sh
pnpm build
```
